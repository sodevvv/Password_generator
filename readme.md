# Password Generator (Python)

Un générateur de mots de passe sécurisé écrit en Python, utilisant le module `secrets`.

Ce projet permet de créer des mots de passe forts, aléatoires et imprévisibles en fonction d’une longueur choisie par l’utilisateur.

---

## Fonctionnalités

- Génération de mots de passe sécurisés
- Longueur personnalisable par l’utilisateur
- Utilisation de lettres (majuscules + minuscules)
- Utilisation de chiffres
- Utilisation de symboles
- Génération cryptographiquement sécurisée (`secrets`)
- Application en ligne de commande (CLI)

---

## Objectif du projet

Ce projet a été réalisé pour apprendre :

- la génération de nombres aléatoires sécurisés en Python
- la différence entre `random` et `secrets`
- les bases de la sécurité des mots de passe
- la création d’un petit outil CLI propre et fonctionnel

---

## ⚙️ Prérequis

- Python 3.12

Aucune bibliothèque externe n’est requise.

## ▶️ Utilisation 
Lancer le programme :

```bash
python main.py
```

Vous pouvez choisir la longueur de votre mot de passe en entrant une valeur numérique.
